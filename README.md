# RecipeMaster Pro 🍳

A comprehensive Android recipe application with video tutorials, ratings, and multi-user roles.

## Features ✨

### Three User Roles:
- **Admin**: Database management, content moderation, user management
- **Chef**: Create recipes with video tutorials, manage own content
- **User**: Browse recipes, watch videos, rate and comment

### Current Features:
- ✅ User Registration & Login
- ✅ Role-based access (User/Chef)
- ✅ Firebase Authentication
- ✅ Cloud Firestore Database
- ✅ Session Management
- ✅ Add Recipes (Chefs)
- ✅ Browse Recipes (Users)
- ✅ My Recipes (Chefs)

### Upcoming Features:
- 📹 Video recipe tutorials
- ⭐ Rating and review system
- 🔍 Advanced search
- 📱 Offline access
- 📋 Shopping list generator
- 👥 Social features

## Tech Stack 🛠️

- **Frontend**: Android (Java/XML)
- **Backend**: Firebase (Auth, Firestore, Storage)
- **UI Components**: Material Design, RecyclerView, CardView
- **Architecture**: MVVM pattern
